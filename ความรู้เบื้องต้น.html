ความรู้เบื้องต้นเกี่ยวกับอินเทอร์เน็ต

ปัจจุบันระบบข่าวสารข้อมูลสารสนเทศระบบเครือข่าย อินเทอร์เน็ตหรือที่เรียกว่าเว็บเพจ มีความสำคัญมากในหน่วยงานต่างๆ ทั้งภาครัฐ เอกชน สถานศึกษา ต่างใช้ช่องทางนี้ติดต่อสื่อสารประชาสัมพันธ์ข้อมูลเกี่ยวกับองค์กรและเป็นที่รวบรวมข้อมูล เพื่อให้ผู้ที่สนใจทำการค้นหาดังนั้น การศึกษา อินเทอร์เน็ตและเว็บเพจจึงมีความสำคัญเป็นอย่างมาก

อินเทอร์เน็ต

หมายถึง เครือข่ายของระบบคอมพิวเตอร์ขนาดต่างๆ เชื่อมต่อเข้าหากันจนเป็นเครือข่ายคอมพิวเตอร์ที่ใหญ่ที่สุด ซึ่งคำว่าอินเทอร์เน็ตมาจากคำว่า "Inter Connected เครื่องคอมพิวเตอร์ทุกเครื่องที่เชื่อมต่อเข้ากับระบบเครือข่ายอินเทอร์เน็ตนั้นสามารถโดยใช้มาตรฐานในการรับส่งข้อมูลหรือที่เรียกว่า โพรโทคอล (Protocol) ในระบบอินเทอร์เน็ตมีชื่อว่า ที่ซีพี/ไอพี (TCP/IP, Transmission Control Protoco//nternet Protoco



ไซเบอร์สเปซ Cyberspace

หมายถึง พื้นที่ช่องว่าง ที่สร้างขึ้นด้วยระบบอิเล็กทรอนิกส์ เพื่อใช้ติดต่อสื่อสาสามารถติดต่อกันได้ทั่วโลกเสมือนท่องไปในอวกาศ มีการนำมาใช้ในชีวิตประจำวันของกลุ่มคนสมาคมที่มีการเชื่อมต่อทางระบบอินเทอร์เน็ตในรูปแบบที่สามารถสื่อสารได้สองทาง มีการโต้ตอบเสมือนพบเจอกันได้ในชีวิตประจำวัน เช่น การรับส่งจดหมายอิเล็กทรอนิกส์ การสื่อสารผ่านสังคมออนไลน์เช่น facebook twitter hi5 อินเทอร์เน็ตจึงเป็นช่องทางหนึ่งที่นิยมแพร่หลายในการติดต่อสื่อผ่านไซเบอร์สเปซ



บริการเครือข่ายสังคม

เป็นรูปแบบเว็บไซต์ในการสร้าง สังคม สำหรับผู้ใช้งานในอินเทอร์เน็ต เขียนอธิบายความสนใจ กิจกรรมที่ได้ทำ และเชื่อมโยงกับความสนใจและกิจกรรมของผู้อื่นในบริการเครือข่ายสังคมมักจะประกอบไปด้วย การแชท (สนทนา) ส่งข้อความ ส่งอีเมล เพลง อัปโหลดรูป การทำงานคือ คอมพิวเตอร์เก็บข้อมูลไว้ในรูปฐานข้อมูล ส่วน video หรือ รูปภาพ อาจเก็บเป็นไฟล์รูปแบบต่างๆ


ระบบชื่อโดเมน DNS : Domain Name System

การติดต่อสื่อสารจำเป็นจะต้องรู้ที่อยู่ของผู้ที่ต้องการติดต่อสื่อสารด้วย เช่น การส่งจดหมายต้องรู้ที่อยู่ของผู้รับและต้องส่งผ่านไปรษณีย์ ซึ่งเป็นตัวกลางในการรับส่ง ดังนั้นวิธีการติดต่อสื่อสารผ่านเครือข่ายอินเทอร์เน็ตก็เช่นเดียวกัน แต่จะมีข้อแตกต่างในเรื่องของวิธีการในการติดต่อสื่อสาร เช่น การติดต่อสื่อสารผ่านจดหมายอิเล็กทรอนิกส์จะต้องมีชื่อ ที่อยู่ และผู้ให้บริการรับส่งข้อมูลดังนี้

- ที่อยู่บนอินเทอร์เน็ต
Internet Address จะประกอบด้วยชื่อของผู้ใช้คอมพิวเตอร์ (User) และชื่อของอินเทอร์เน็ต (Internet Name)โดยจะมีรูปแบบคือ ชื่อผู้ใช้@ชื่อของอินเทอร์เน็ต user@domain.xxx ตัวอย่างเช่น maythasit@hotmal.co.th


- ระบบชื่อโดเมน
ระบบของชื่อที่ใช้เรียกแทนการเรียกหมายเลขอินเทอร์เน็ตหรือหมายเลขไอพี (IP Address) เนื่องจากหมายเลขไอพีมีความยุ่งยากสำหรับการจดจำ ทำให้ขณะที่ใช้งานระบบเครือข่ายอินเทอร์เน็ตจะทำให้เกิดความยุ่งยาก จึงได้มีการพัฒนานำเอาระบบชื่อที่เป็นตัวอักษรมาใช้แทนหมายเลขไอพี จึงมีวิธีการตั้งชื่อให้จดจำและใช้งานง่าย


- หมายเลขอินเทอร์เน็ต
IP Address จะเป็นรหัสประจำตัวของคอมพิวเตอร์ที่ต่อเข้ากับอินเทอร์เน็ต โดยหมายเลขนี้จะมีรหัสไม่ซ้ำกัน ประกอบไปด้วยตัวเลขจำนวน 4 ชุด โดยแต่ละชุดจะมีค่าตัวเลขอยู่ระหว่าง 0 - 255 และคั่นด้วยเครื่องหมายจุด () เช่น 127.0.0.1


- ชื่ออินเทอร์เน็ต
Domain Name System : DNS เป็นชื่อที่อ้างถึงคอมพิวเตอร์ที่ต่อเข้ากับอินเทอร์เน็ต เมื่อต้องการจะสื่อสารกับคอมพิวเตอร์เครื่องอื่น เครื่องนั้นก็จะทำการสอบถามหมายเลข IP ของเครื่องที่ต้องการกับ DNS Server ซึ่งจะทำการค้นหาหมายเลขในฐานข้อมูล DNS ช่วยแปลงหมายเลขไอพีซึ่งเป็นชุดตัวเลขที่จดจำยาก เช่น ถ้าต้องจดจำตัวเลข72.30.2.43 อาจจดจำได้ยาก หากเปลี่ยนเป็นชื่อเป็น yahoo.com ผู้ใช้จะสามารถจดจำได้ง่ายกว่า

การบริการของอินเทอร์เน็ต

ในอินเทอร์เน็ตมีบริการต่างๆ แบ่งอออกตามจุดประสงค์การใช้งาน เช่น การสื่อสาร หาข้อมูล นำเสนอสินค้า เครือข่ายอินเทอร์เน็ตที่เชื่อมกันอยู่นั้น ประกอบด้วยเครื่องคอมพิวเตอร์ที่ทำหน้าที่เป็น Server ทำหน้าที่ต่างกันในการให้บริการ แบ่งเป็น บริการด้านการสื่อสารและบริการด้านการค้นหาข้อมูล

- การบริการของอินเทอร์เน็ต

จดหมายอิเล็กทรอนิกส์์

E-mal ได้รับความนิยมมาก ใช้งานที่ง่ายและความต้องการในการติดต่อของผู้คนในปัจจุบันมีสูง ทำให้ได้รับการตอบรับจากผู้ใช้มาก E-mal สามารถส่งข่าวสารไปถึงผู้รับได้ในระยะเวลาอันสั้นและรวดเร็ว เป็นระบบที่รวดเร็วกว่าการสื่อสารแบบอื่นและประหยัดเงินในการใช้งานมากกว่าการสื่อสารแบบเก่า

การค้นหาข้อมูล

Search Engine เว็บไซต์ที่ให้บริการค้นหาข้อมูล รวบรวมข้อมูล มีการจัดแยกออกเป็นหมวดหมู่ รูปภาพ วิดีโอเพื่อความสะดวก เพียงข้อความในช่อง กดปุ่ม Enter จะปรากฎข้อมูลอย่างย่อๆ และเว็บไซต์ที่เกี่ยวข้องสามารถเข้าไปศึกษาเพิ่มเติมได้ เว็บไซต์ที่ให้บริการ เช่น yahoo.com, google.com

ธุรกิจออนไลน์

Online Business การค้าขายผ่านระบบคอมพิวเตอร์ไม่จำเป็นต้องมีพนักงานขาย กระแสของ E-commerce ได้แพร่หลายทั่วโลก แนวโน้มในอนาคต Online Business จะเป็นแนวทางที่เกือบทุกองค์กรจะนำมาใช้เพื่อเพิ่มช่องทางของธุรกิจและปรับตัวให้ทันกับการทำธุรกิจในยุคปัจจุบัน

การโอนถ่ายข้อมูลระหว่างเครื่อง

FTP มีความจำเป็นมาก เนื่องจากมีการโอนถ่ายข้อมูลระหว่างเครื่องมากขึ้น เช่น เมื่อออกแบบเว็บไซต์เสร็จแล้ว ต้องมีการส่งข้อมูลทั้งหมดไปที่ Server เพื่อให้ผู้ใช้ User สามารถดูข้อมูลได้ ปัจจุบันมีโปรแกรมที่สามารถทำให้ใช้งานในรูปแบบของ FTP ได้ง่ายขึ้น เช่น WS_ FTP,Cute FTP เป็นต้น

World Wide Web

WWW ได้รับความนิยมรองจาก E-mail เนื่องจากเเป็นช่องทางการเข้าถึงข้อมูลที่สะดวกความยมของอินเทอร์เน็ตที่เกิดขึ้นนั้น ส่วนหนึ่งเป็นเพราะเว็บไซต์ เนื่องจากมีการเกิดขึ้นใหม่เสมอ ความนิยมของเว็บไซต์ทำให้เกิดธุรกิจที่เกี่ยวข้อง เช่น E.commerce การเจริญเติบโตของตลาดคอมพิวเตอร์

การศึกษาระยะทางไกล

WLM หรือ Instant Messenger โปรแกรมส่งข้อความข้ามระบบเครือข่ายแบบทันที ส่งข้อความ วิดิโอเสียง ไฟล์ผ่านทาง WLM การทำงานคล้ายกับโปรแกรม ICQ แต่โปรแกรม MSN Messenger นั้นผู้ใช้ E-mail Address ของ Hotmail สามารถใช้ MSN ได้ทันที รวมถึงมีการเชื่อมโยง E-mail และ facebook

บริการกระดานข่าวอิเล็กทรอนิกส์

Usenet มีการแบ่งหัวข้อความสนใจเรื่องต่างๆ แล้วแต่จะมีคนตั้งขึ้น ผู้ที่เข้ามาอ่านสามารถแสดงความคิดเห็นได้ เรียกอีกชื่อหนึ่งว่า Newsgroup หรือ Community Webboard เว็บไซต์ที่ให้การบริการในประเทศไทย เช่น เว็บไซต์พันทิป สามารถเข้าไปสอบถามปัญหาหรือเรื่องราวต่างๆ ที่สนใจได้



จำนวนผู้ใช้อินเทอร์เน็ตในประเทศไทย

ปี 2551 ตัวเลขของสำนักงานสถิติแห่งชาติอยู่ที่ 10.96 ล้านคน (ที่มา-สำนักงานสถิติแห่งชาติ 2551)
ปี 2552 ตัวเลขของต่างประเทศอยู่ที่ 20.03 ล้านคน (ที่มา - UN E-Government Survey : 2009)
ปี 2552 ตัวเลขของ NECTEC อยู่ที่ 18.3 ล้านคน (ที่มา-ศูนย์เทคโนโลยีอิเล็กทรอนิกส์และคอมพิวเตอร์แห่งชาติ NECTEC )
ปี 2553 ตัวเลขผู้ใช้อินเทอร์เน็ตของ NECTEC อยู่ที่ 24 ล้านคน (ที่มา-เป็นตัวเลขประมาณการณ์จากการเก็บข้อมูล Unique IP ที่วิ่งบน Traffic)


ปัจจัยที่ทำให้ผู้ใช้งานอินเทอร์เน็ตเพิ่มมากขึ้น

1. โทรศัพท์เคลื่อนที่ ประเภท Smart Phone มีการพัฒนาให้รองรับการให้บริการ Moboile Internet เข้าถึงอินเทอร์เน็ตได้ทุกที่ทุกเวลา
2. แนวโน้มราคาของอุปกรณ์ที่รองรับการใช้งานอินเทอร์เน็ตความเร็วสูงที่ปรับตัวลดลง เช่น Notebook Router และ Air Card เป็นต้น
3. Widget/Gadget สร้างกระแสความตื่นตัวในการใช้งาน Application ใหม่ๆ เช่น iPad ของ Apple กระตุ้นความต้องการใช้งานอินเทอร์เน็ตความเร็วสูง
4. ความนิยมในการใช้งาน Social networking ทำให้ผู้ใช้อินเทอร์เน็ตต้องการความสะดวกรวดเร็วในการดาวน์โหลด/อัพโหลดข้อมูล
5. Application การใช้งานแบบ Real-Time การให้บริการ Video Call และเทคโนโลยี Virtualization เกมออนไลน์ Social Network Game
6. นโยบายภาครัฐ จะต้องมีการพัฒนาโครงข่ายอินเทอร์เน็ตความเร็วสูง และต้องให้ประชาชนเข้าถึงอินเทอร์เน็ตความเร็วสูงได้อย่างทั่วถึง
