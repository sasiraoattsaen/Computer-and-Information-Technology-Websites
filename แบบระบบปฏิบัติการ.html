แบบทดสอบ ระบบปฏิบัติการ
คำชี้แจง : จงเลือกคำตอบที่ถูกต้องที่สุด
1) เครื่องคอมพิวเตอร์ต้องมีซอฟต์แวร์ใดจึงจะทำงานได้
ก) Microsoft Office
ข) Microsoft Windows
ค) Ubuntu
ง) ข้อ ข และ ค ถูก

2) Microsoft Windows รุ่นใดเป็นรุ่นล่าสุด
ก) Windows Me
ข) Windows XP
ค) Windows 11
ง) Windows 8

3) ข้อใดไม่ใช่ระบบปฏิบัติการ
ก) Windows 8
ข) Ubunti
ค) ios
ง) ไม่มีข้อถูก

4) แถบทางด้านล่างของหน้าต่างหมายถึงข้อใด
ก) Menu bar
ข) Taskbar
ค) Title bar
ง) Scroll bar

5) การปักหมุดให้กับโปรแกรมทำเพื่อประโยชน์อะไร
ก) ทำให้เห็นได้ง่ายขึ้น
ข) ทำงานได้เร็วขึ้น
ค) เรียกใช้โปรแกรมได้ง่านขึ้น
ง) หยุดโปรแกรมไม่ให้เคลื่อนที่

6) Jump List มีประโยชน์อย่างไร
ก) เปิดไฟล์งานเดิมได้เริ่วขึ้น
ข) สลับหน้าต่างโปรแกรม
ค) ย้อนการทำงานของไฟล์เอกสาร
ง) มีหน้าต่าง Preview ให้ดูตัวอย่างงาน

7) คำสั่งข้อใดใช้สำหรับย้ายไฟล์
ก) Ctrl + X กับ Ctrl + V
ข) Alt + C กับ Alt + V
ค) Alt + X กับ Alt + V
ง) Alt + C กับ Ctrl + V

8) ข้อใดเป็นวิธีการสร้างโฟลเดอร์ใหม่
ก) คลิกเมาส์ขวาแล้วลากไปตำแหน่งที่ต้องการ
ข) คลิกเมาส์ซ้ายเลือกเมนู New Folder
ค) คลิกเมาส์ซ้ายเลือกเมนู New
ง) คลิกเมาส์ขวาเลือกเมนู New จะปรากฎเมนูย่อยอีกเลือก Folder

9) ข้อใดใช้สำหรับเปลี่ยนชื่อไฟล์หรือโฟลเดอร์
ก) F1
ข) F2
ค) F3
ง) F4

10) ข้อใดเป็นการเลือกไฟล์ที่ไม่ติดกัน
ก) Ctrl + A
ข) คลิกเลือกไฟล์ในหน้าต่าง แล้วกดปุ่ม Organize > Select all
ค) คลิกเลือกไฟล์แรก กดปุ่ม Shift คลิกที่ไฟล์สุดท้ายสุดท้าย
ง) กดปุ่ม Ctrl คลิกเลือกไฟล์ที่ต้องการ
