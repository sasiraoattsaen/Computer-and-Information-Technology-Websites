ระบบปฏิบัติการ

ระบบปฏิบัติการ (OperatingSystem : Os) โปรแกรมควบคุมการทำงานของเครื่องคอมพิวเตอร์ เช่น การแสดงผล ควบคุมข้อมูลต่างๆ ในการติดต่อกับผู้ใช้ เป็นสื่อกลางระหว่างผู้ใช้กับเครื่องคอมพิวเตอร์ให้สามารถสื่อสารกันได้ ควบคุมและจัดสรรทรัพยากรให้กับโปรแกรมต่างๆ ที่ถูกติดตั้งอยู่ในคอมพิวเตอร์

ระบบปฏิบัติการ Mac OS

เป็นระบบปฏิบัติการที่ใช้สำหรับเครื่อง Macintosh ของบริษัท Apple เหมาะสำหรับผู้ที่ทำงานเกี่ยวกับการออกแบบทางกราฟฟิก เป็นระบบปฏิบัติการสำหรับเครื่องคอมพิวเตอร์ Macintosh โดยเฉพาะ ระบบ macOS นิยมใช้ในงานที่ต้องการความละเอียดของรูปภาพสูง เช่น งานโฆษณา งานเกี่ยวกับภาพยนตร์ และงานที่ต้องอาศัยการคำนวณชั้นสูงและการออกแบบ เช่น ออกแบบสิ่งพิมพ์ เครื่องยนต์ และแบบอาคาร และอื่นๆ


ระบบปฏิบัติการ Linux

Linux เป็นระบบปฏิบัติการประเภทฟรีแวร์ (Freeware) คือไม่เสียค่าใช้จ่ายในการซื้อ ทำให้การขยายตัวของ Linux นั้นเป็นไปอย่างรวดเร็วสามารถนำไปใช้สำหรับทำงานเป็น Server ขนาดใหญ่ทำงานได้บนเครื่องพีซีทั่วไป จึงเป็นระบบปฏิบัติการที่มีความต้องการทรัพยากรของระบบในขั้นต่ำในปัจจุบันพัฒนามาเป็นระบบปฏิบัติการ Ubuntu


ระบบปฏิบัติการ Microsoft Windows

Windows เป็นระบบปฏิบัติการที่พัฒนาโดยบริษัท Microsoft การติดต่อหรือการป้อนคำสั่งระหว่างผู้ใช้และเครื่องคอมพิวเตอร์ (User interface) จะเป็นแบบกราฟิกหรือใช้รูปภาพแทนคำสั่งเรียกว่า GUI (Graphic User Interface)Windows ได้รับความนิยมมากที่สุด มีลักษณะการใช้งานที่ง่าย สนับสนุน Multimedia อย่างสมบูรณ์ ไม่ว่าจะดูหนัง ฟังเพลง เล่นเกมมีการพัฒนามาตลอดโดยออกเป็นเวอร์ชั่นต่างๆ เช่น Windows 95, Windows 98, Windows Me, Windows XP,Windows Vista, Windows 7


การใช้งานระบบปฏิบัติการ Windows 11 เบื้องต้น

เป็นระบบปฏิบัติการวินโดวส์รุ่นใหม่ ที่พัฒนาโดยไมโครซอฟท์ เปิดตัวอย่างเป็นทางการเมื่อวันที่ 24 มิถุนายน 2021 มีกำหนดอัปเดตระบบให้ผู้ใช้งานอุปกรณ์วินโดวส์ 10 ที่รองรับการใช้งานโดยไม่คิดมูลค่าอย่างเป็นทางการ

ส่วนประกอบในหน้าจอของ Windows 11

เมื่อเปิดเครื่องคอมพิวเตอร์ขึ้นมาจะพบกับหน้าจอที่เรียกว่า เดสก์ท็อป (Desktop) ผู้ใช้สามารถวางสิ่งต่างๆ ไม่ว่าจะเป็นไฟล์ข้อมูล โปรแกรมสำหรับใช้งาน และสิ่งตกแต่งอื่นๆ มากมายไว้บน Desktop



1. Start เป็นจุดเริ่มต้นสำหรับการเปิดโปรแกรม
2. Taskbar คือ แถบยาวๆ ที่วางอยู่ด้านล่างของหน้าจอ ใช้แสดงชื่อเรียกใช้โปรแกรมที่กำลังเปิดอยู่
3. Icon ปุ่มไอคอนเป็นสัญลักษณ์แทนโปรแกรมผู้ใช้สามารถเปิดโปรแกรมได้ด้วยการดับเบิลคลิกไอคอน
4. Notifcation area เป็นส่วนที่ Windows ใช้บอกสถานการณ์ทำงานต่างๆ กับผู้ใช้งาน เช่น วันเวลา,การเตือน
5. Desktop เป็นพื้นที่หน้าจอที่ใช้ทำงานกับWindows 11 ประกอบด้วยทาสก์บาร์ปุ่มและไอคอนหรือหน้าต่างโปรแกรมที่ถูกใช้งานวางอยู่บนเดสก์ท็อป


การปักหมุดโปรแกรมที่ใช้งานบ่อย ๆ
- การปักหมุดไว้ที่เมนู Start
คลิกเมาส์ขวาของโปรแกรมที่ต้องการปักหมุด จะปรากฏเมนูย่อยขึ้นมาให้เลือก Pin to Start การยกเลิกการปักหมุดให้คลิกขวาที่ไอคอนของโปรแกรมเลือกคำสั่ง Unpin from Start


- การปักหมุดไว้ที่ Taskbar
คลิกเมาส์ขวาของโปรแกรมที่ต้องการปักหมุด จะปรากฎเมนูขึ้นมาให้เลือก More จะปรากฏเมนูย่อยขึ้นมาให้เลือก Pin to Taskbar การยกเลิกการปักหมุดให้คลิกขวาที่ไอคอนของโปรแกรมเลือกคำสั่ง Unpin from taskbar


การเข้าถึงไฟล์ด้วย Jump List
การเปิดไฟล์ที่ใช้งานมาก่อนอย่างรวดเร็วช่วยให้ประหยัดดเวลาในการทำงาน โปรแกรมจะแสดง รายการไฟล์งานที่ทำไว้ล่าสุด เพื่อเรียกใช้งานได้อย่างรวดเร็ว คลิกเมาส์ขวาของโปรแกรมที่ต้องการเรียกงานขึ้นมาทำต่อหรือแก้ไข จะปรากฎรายการไฟล์งานที่ทำไว้ล่าสุดขึ้นมาให้เรียกใช้งาน


การแสดงตัวอย่างด้วย Taskbar Thumbnail
เมื่อผู้ใช้งานเปิดโปรแกรมขึ้นมาใช้งานหลายโปรแกรมหรือเปิดโปรแกรมเดียวกันหลายหน้าต่าง เช่น เปิดไฟล์เอกสารเวิร์ดขึ้นมาใช้งานพร้อมกัน 3 ไฟล์ เมื่อเลื่อนเมาส์ผ่านโปรแกรมบนทาสก์บาร์ Windows 7 จะแสดงหน้าต่าง Preview ขึ้นมาให้เห็นหน้าต่างของงานขนาดเล็กเพื่อสะดวกในการสลับการทำงาน และถ้าต้องการเลือกทำงานกับโปรแกรมหรือไฟล์งานใด ก็ให้คลิกเมาส์ที่หน้าต่าง Thumbnail หน้าต่างก็จะขยายเต็มจอทันที



การจัดการไฟล์และโฟลเดอร์
ไฟล์
สิ่งที่ใช้ในการอ้างอิงข้อมูล หรือโปรแกรมที่เก็บบันทึกลงในฮาร์ดดิสก์ ปกติการอ้างอิงไฟล์จะทำโดยผ่านชื่อไฟล์และตามด้วยจุดนามสกุลของโปรแกรมที่ถูกสร้างขึ้น จำนวนของไฟล์นั้นจะขึ้นอยู่กับขนาดของไฟล์และหน่วยความจำที่ใช้ในการเก็บการเข้าถึงแหล่งข้อมูลหรือสื่อบรรจุข้อมูลชนิดต่างๆ ที่เชื่อมต่อกับคอมพิวเตอร์จะต้องอาศัยหน้าต่าง Computer หรืออาจเรียกว่า Windows Explorer ในหน้าต่างจะแสดงชื่อและรายละเอียดของอุปกรณ์จัดเก็บข้อมูล

โฟลเดอร์
เปรียบเหมือนแฟ้มเอกสารที่ใช้แบ่งไฟล์ออกเป็นหมวดหมู่ เพื่อความสะดวกในการ ค้นหาไฟล์ ภายในโฟลเดอร์สามารถสร้างโฟลเดอร์ย่อยเพื่อแบ่งกลุ่มไฟล์ย่อยๆ ลงไปอีกได้การตั้งชื่อไฟล์หรือโฟลเดอร์สามารถมีอักขระได้มากที่สุด 255 อักขระรวมถึงช่องว่างระหว่างอักขระด้วย แต่ไม่สามารถมีอักขระต่อไปนี้คือ: * ? " < / ในการตั้งชื่อไฟล์หรือโฟลเดอร์

- การสร้างโฟลเดอร์ใหม่ด้วยเมนู คลิกที่ปุ่ม New จะปรากฎเมนูขึ้นมาให้เลือก Folder


- สร้างโฟลเดอร์ใหม่ด้วยการคลิกเมาส์ขวา คลิกเมาส์ขวาในหน้าต่างที่ต้องการสร้างโฟลเดอร์ใหม่ จะมีเมนูย่อยให้เลื่อนเมาส์ไปที่ New จะปรากฏเมนูย่อยขึ้นอีกให้เลือก Folder


- เปลี่ยนชื่อไฟล์หรือโฟลเดอร์ คลิกที่ไฟล์ที่ต้องการเปลี่ยนชื่อ กดปุ่ม F2 หรือ คลิกปุ่ม Rename


- การเลือกไฟล์หลายๆไฟล์ที่อยู่ติดกันนั้น ทำได้โดยคลิกที่ชื่อไฟล์แรกที่ต้องการเลือก หลังจากนั้นกดปุ่ม Shift ค้างไว้ และคลิกที่ชื่อไฟล์สุดท้ายของกลุ่มไฟล์ที่จะเลือก


- การเลือกไฟล์หลายๆ เมื่อไฟล์ที่อยู่ไม่ติดกัน สามารถทำได้โดยการกดปุ่ม Ctr! ค้างไว้ และคลิกที่ชื่อไฟล์ที่ต้องการเลือกทีละไฟล์จนครบหมดทุกไฟล์


- การคัดลอกหรือย้ายไฟล์ คลิกเลือกไฟล์หรือโฟลเดอร์ คลิกเมาส์ค้างแล้วลากไฟล์หรือโฟลเดอร์ไปยังที่ใหม่


- การลบไฟล์หรือโฟลเดอร์ ถ้าต้องการลบไฟล์หรือโฟลเดอร์ที่ไม่ใช้งานเพื่อป้องกันการสับสน ให้คลิกที่ไฟล์หรือโฟลเดอร์ ที่ต้องการลบ จากนั้นให้กดปุ้ม Delete หรือคลิกเมาส์ขวาจะปรากฏเมนูย่อยให้เลื่อนไปคลิกที่ Delete เมื่อลบแล้วไฟล์หรือโฟลเดอร์จะถูกลบไปเก็บไว้ที่ Recycle Bin กรณีที่ต้องการลบไฟล์หรือโฟลเดอร์แบบถาวรให้เลือกไฟล์หรือโฟลเดอร์ที่ต้องการลบ จากนั้นให้กดปุ่ม Shift พร้อมปุ่ม Delete ไฟล์จะถูกลบออกไปจากเครื่องและไม่สามารถกู้คืนมาได้อีก


- การทำความสะอาดถังขยะ การลบข้อมูลจากเครื่องลง Recycle Bin ไม่ได้เป็นการลบไฟล์เหล่านั้นออกจากเครื่องคอมพิวเตอร์ ข้อมูลยังอยู่ใน Hard Disk ถ้าในถังขยะเป็นข้อมูลที่แน่ใจว่าไม่ต้องการแล้วก็ควรลบออกจากถังขยะ เพราะจะทำให้มีพื้นที่ใน Hard Disk เพิ่มขึ้น ดับเบิลคลิกที่ไอคอน Recycle Bin จะปรากฏหน้าต่างขึ้นมา คลิกเลือกที่ปุ่ม Empty the Recycle Bin จะปรากฎหน้าต่างให้ตอบยืนยันการลบให้คลิกที่ปุ่ม YES ไฟล์จะถูกลบแบบถาวร หรือ คลิกเมาส์ขวาที่ไอคอน Recycle Bin จะปรากฏเมนูย่อยให้คลิกเลือก Empty Recycle Bin
